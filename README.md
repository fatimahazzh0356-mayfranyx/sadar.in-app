# sadar.in-app
Sadar.in adalah aplikasi edukasi interaktif berbasis Flutter yang membantu pengguna memahami kekerasan seksual non-verbal, menjaga ruang aman, dan mengenali batas personal. Dilengkapi materi edukasi, ilustrasi interaktif, kuis pemahaman, serta akses hotline bantuan resmi dengan desain modern dan ramah pengguna.
