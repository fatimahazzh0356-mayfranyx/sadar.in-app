# Sadar.in

Sadar.in adalah aplikasi edukasi interaktif berbasis Flutter yang membantu pengguna memahami kekerasan seksual non-verbal, menjaga ruang aman, dan mengenali batas personal.

## ✨ Fitur Utama
- Materi edukasi interaktif
- Ilustrasi visual edukatif
- Kuis pemahaman
- Sistem badge & high score
- Hotline bantuan resmi
- UI modern dan responsif

## 🛠 Teknologi
- Flutter
- Dart

## 🎯 Tujuan Aplikasi
Meningkatkan kesadaran pengguna mengenai pelecehan non-verbal serta memberikan akses edukasi dan bantuan yang mudah dijangkau.

## 📱 Tampilan Aplikasi
Aplikasi memiliki desain modern dengan nuansa warna pink pastel yang nyaman dan ramah pengguna.
